```
1XX ---临时相应
100--表示请求者应继续发起请求
101---请求者要求服务端切换协议，服务端已确认并且准备切换
2XX--表示成功处理请求的状态码
200---表示服务端成功处理了请求
201---请求成功并且服务端创建了新的资源
202---服务端接受请求但尚未处理
203---服务端已经处理了请求但返回的信息可能是另一来源
204---服务端成功处理请求但并没有返回任何信息
205---同上
206---服务端处理了部分get请求
3XX重定向
300---服务端可根据请求者选择一项进行操作
301---重定向 资源永久变更
302---服务端重定向到别的资源
303---请求者应对不同位置使用单独的请求来检索
304---使用了协商缓存
305---使用代理来请求网页
306---临时重定向
4XX---请求错误
400---客户端语法错误
401---未授权
403---服务器拒绝
404---找不到资源
405--该方法禁用
406--不接受
407--需要代理授权
408--服务端等待请求超时
409---发送冲突
5XX---服务端错误
500---服务端内部错误
501--尚未实施
502--网关错误
503--服务器不可用
504--网关超时
```
