###作用域
```
作用域是程序源代码中定义变量的一个区域  
作用域决定了如何查找变量也就是确定了当前执行代码对变量的访问权限  
JS采用词法作用域(静态作用域)
```
###作用域链
```
当我们查找一个变量的时候会先从当前上下文的变量对象中查找如果木有就到父级上下文的函数对象中查找一直找到全局上下文的变量对象
这种由多个执行上下文构成的链表就叫作用域链  
也可以说一个函数创建的时候,它的作用域链就已经确定了,这个主要是因为函数有一个scope属性,当函数创建的时候scope属性会把当前作用域链上的  
所有变量对象存储起来 当为上下文创建它的作用域链 Scope=AP.concat([scope]);
```
